# TinyApp Project

TinyApp (rebranded as TeenyApp) is a full stack web application built with Node, Express and Bulma.css that allows users to shorten long URLs (Ã  la bit.ly).


## Salient Technical Features

- Use method-overide to use REST PUT & DELETE APIs and proper route handling in server side logic.
- Inbuilt page analytics using cookies for each individual visitor, logging all visits & displaying both total & unique visits for each generated Short URL.

## Dependencies

NPM

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- method-override

CDN:

- Bulma.css

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
